## 📝 推奨アクション

### 1. Sync Jira Data の修正

GitHub Secretsに必要な環境変数を設定してください：

```bash
gh secret set CONFLUENCE_BASE_URL --body "https://giginc.atlassian.net"
gh secret set CONFLUENCE_USER_EMAIL --body "kanri@jukust.jp"
gh secret set CONFLUENCE_SPACE_KEY --body "CLIENTTOMO"
gh secret set JIRA_BASE_URL --body "https://giginc.atlassian.net"
gh secret set JIRA_USER_EMAIL --body "kanri@jukust.jp"
```

### 2. Tests ワークフローの有効化

`test.yml`ワークフローをGitHubにプッシュして有効化してください：

```bash
git add .github/workflows/test.yml
git commit -m "Add test workflow for CI/CD"
git push origin main
```

### 3. ワークフローの実行確認

プッシュ後、以下のコマンドでワークフローの実行状況を確認できます：

```bash
# ワークフロー一覧を確認
gh workflow list

# 最新の実行履歴を確認
gh run list --limit 10

# 特定のワークフローの実行履歴を確認
gh run list --workflow="Tests" --limit 5
```
